## 🔄 CROSS-SKILL CONFLICTS (Resolution Hierarchy)

### Color/Font Conflicts
```
WCAG Contrast (Layer 8) > design-taste-frontend > figma-design aesthetic
If accessibility is needed → WCAG WINS (accessibility is non-negotiable)
```

### UX Law Conflicts
```
PRIORITY 1 — Safety & Error Prevention (overrides everything)
PRIORITY 2 — User Control & Trust
PRIORITY 3 — Usability (core task completion)
PRIORITY 4 — Aesthetics & Delight (polish)
```

### Platform Conflicts
```
iOS HIG / Android Material 3 / Web Standards > Generic advice
Always check platform rules first
```
