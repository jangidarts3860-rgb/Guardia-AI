# Final Design QA v2 — Pre-Delivery Gate

> **Run this checklist BEFORE every design handoff. Zero exceptions.**

---

## VISUAL

- [ ] Correct breakpoints tested (375 / 768 / 1024 / 1440)?
- [ ] Dark mode works without hardcoded colors?
- [ ] No emojis as icons (use SVG/Phosphor/Lucide)?
- [ ] Icon style consistent throughout?
- [ ] No lorem ipsum anywhere?

---

## INTERACTION

- [ ] All tap targets ≥44px? (56px for India Tier 2/3)
- [ ] Loading state on ALL async actions?
- [ ] Error states built (not just happy path)?
- [ ] Empty states designed?
- [ ] Back/cancel always available?

---

## ACCESSIBILITY

- [ ] Contrast 4.5:1 on all text? (WCAG AA)
- [ ] Labels on all inputs?
- [ ] Color + icon + text on errors (not color alone)?
- [ ] Focus order logical?
- [ ] prefers-reduced-motion handled?

---

## PERFORMANCE

- [ ] Images lazy loaded below fold?
- [ ] No layout shift on load?
- [ ] Animations on transform/opacity only?

---

## UX LAW COMPLIANCE (from ux-thinking)

- [ ] Severity Critical issues addressed?
- [ ] No anti-patterns from Layer 7?
- [ ] Layer 8 (WCAG) fully passed?

---

## HANDOFF PACKAGE

- [ ] Figma file structure clean (pages named, frames named)
- [ ] Component variants complete (all states)
- [ ] Design tokens exported (JSON/CSS)
- [ ] Annotated specs with spacing/typography/color tokens
- [ ] Prototype flows linked (happy path + error paths)
- [ ] Developer notes on complex interactions

---

## EXTERNAL SKILL VALIDATION (For Tier 2 External Outputs)

- [ ] All code/component specs use core design-tokens (no hardcoded raw hex/RGB/HSL or absolute pixel values)?
- [ ] All touch targets in generated code/mockups conform to core accessibility rules (≥44px / 56px India)?
- [ ] Colors generated by external libraries pass WCAG 4.5:1 via core accessibility guidelines?
- [ ] All animations and easing functions map to core motion-tokens (durations/easing limits)?
- [ ] No inline styling or variables that bypass the core token system?
- [ ] Indian market patterns (UPI structure, bilingual layout spacing, offline cues) respected?
- [ ] Target platform guidelines (iOS HIG / Android Material 3 / Web margins) strictly applied?

---

**ANY UNCHECKED BOX = DO NOT HAND OFF. COMPLETE IT FIRST.**
